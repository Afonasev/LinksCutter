## Backend

### Installing deps

    pip install -r requirements.txt

### Running debug server

    python manage.py run

### Running the testsuite

The minimal requirement for running the testsuite is ``pytest``.  You can
install it with:

    pip install pytest

Then you can run the testsuite with:

    py.test

## Frontend

### Installing deps

    cd ./frontend
    npm i
